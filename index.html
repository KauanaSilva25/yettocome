Qual a diferença entre Servers e Clients?

Servers

São servidores responsáveis pelo armazenamento,recebimento e fornecimento de infomações ao cliente*

Um servidor pode ser uma rede local, ou uma rede compartilhada.

Clients*

São dispositivos que consomem informações de uma servidor, como por exemplo: celular, tablet, computador

Para quê servem os navegadores?

Navegadores ou em inglês browsers são programas utilizados para abrir páginas de internet, ou como são mais conhecido Sites.

Existem vários tipos de navegadores disponíveis, apesar de terem a mesma função, alguns navegadores possuem algumas diferenças em ícones, barras e gerenciamento.

Atualmente os Nvegadores não estão disponíveis apenas para computadores mas para vários dispositivos móveis.

O que são tags utilizadas no HTML?

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere, são exemplos de tag: class, id, title, hiden, il, ol entre outras.3
